PartitionStatus
===============

Java编写的用于获取磁盘分区状态的小工具，可以生成Excel或者HTML.


依赖
===============

* JDK1.6+


适用环境
===============

* Linux系统


目录说明
===============

> getPartitionStatus.jar  ---- 主程序

> start.bat   ---- windows系统上运行主程序的脚本

> start.sh   ---- Linux系统上运行主程序的脚本

> conf      ---- 为配置文件目录

> lib       ---- 依赖库文件目录

> result    ---- 生成的结果存放目录


注意事项
===============

* 所有主机的用户名和密码需相同
